# BestBuy

## Overview
**BestBuy** is a custom tool designed to extract and identify prices from various websites using **WordPress**. By integrating **WP Scraper** and custom scripts, the tool automates web scraping, ensuring accurate and up-to-date price detection for e-commerce and market analysis.

## Features
- Automated web scraping for price tracking
- Integration with **WP Scraper** for efficient data extraction
- Supports **dynamic website structures** for accurate results
- Real-time price updates
- User-friendly interface for seamless navigation

## Technologies Used
- **WordPress**
- **WP Scraper Plugin**
- **Custom JavaScript & PHP scripts**
- **MySQL** (for data storage and management)

## Installation
To set up **BestBuy** on your WordPress site, follow these steps:

1. **Clone the repository**
   ```sh
   git clone https://github.com/your-username/bestbuy.git
   cd bestbuy
   ```

2. **Install WordPress and WP Scraper**
   - Ensure you have a running WordPress installation.
   - Install the **WP Scraper** plugin from the WordPress plugin store.

3. **Upload Custom Scripts**
   - Navigate to your WordPress theme folder.
   - Place the provided **custom scripts** in the appropriate directory.

4. **Configure MySQL Database**
   - Import the provided SQL file to create the necessary tables.
   - Update database credentials in the config file.

5. **Run the BestBuy Tool**
   - Access the tool via your WordPress admin panel.
   - Enter website URLs to start scraping and price tracking.

## Usage
1. Log in to your WordPress dashboard.
2. Navigate to the **BestBuy** panel.
3. Enter product URLs for price extraction.
4. View and analyze extracted prices in the dashboard.
